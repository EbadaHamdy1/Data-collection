Web Scraping Project: Popular Programming Languages and HTML Color Codes
Overview

This is an educational project demonstrating web scraping using Python.
It scrapes data from multiple web pages using requests and BeautifulSoup, processes it into structured data, and saves the results in CSV format.

The project covers:

Extracting links (<a>) and images (<img>) from a web page.

Scraping HTML tables for color codes and programming language data.

Cleaning and organizing data using Pandas.

Saving the final dataset as a CSV file.

Libraries Used

requests – for sending HTTP requests and fetching web page content.

BeautifulSoup (from bs4) – for parsing HTML content.

pandas – for creating DataFrames and saving data as CSV.
